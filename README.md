# infra-automation

This repository contains infrastructure automation and administrative tooling built primarily with PowerShell and Python. The focus is on identity, device management, API integrations, and operational efficiency.

## Scope

Scripts and utilities in this repository are designed to support real-world systems administration and IT operations, including:

- Identity and access management (Entra ID / Azure AD)
- Device and endpoint management (Intune, MDM-related tooling)
- API integrations and data normalization
- Reporting, cleanup, and operational automation

## Technologies

- PowerShell
- Python
- REST APIs
- JSON and CSV data handling

## Structure

## Usage Notes

- Scripts are written to be readable, reusable, and safe by default.
- Configuration such as tokens, secrets, or environment variables must be supplied externally.
